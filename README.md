# Python Script Obfuscation with PyArmor

## Description
This guide provides steps to obfuscate a Python script using PyArmor and package it into an executable with PyInstaller.

## Requirements
- Python 3.x
- `pyarmor==7.6.0`
- `pyinstaller`
